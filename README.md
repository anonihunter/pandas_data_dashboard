# 📊 Pandas Data Dashboard

An interactive data visualization dashboard built using Streamlit, Pandas, and Matplotlib. This tool allows users to upload CSV files and generate meaningful insights through dynamic charts.

---

## 🚀 Features

- 📁 Upload CSV files بسهولة
- 🔍 Dataset preview
- 📊 Bar Chart visualization
- 📈 Line Chart visualization
- 🥧 Pie Chart visualization
- 🎯 Dynamic column selection
- ⚠️ Error handling for invalid files

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly.Express
